# moyenne-notes
Mini projet Python pour apprendre GitHub.
Programme qui calcule la **somme** et la **moyenne** d’un tableau de notes prédéfini,  
et qui affiche une appréciation selon la moyenne.

## Fonctionnalités
- Affiche la liste des notes
- Calcule la somme des notes
- Calcule la moyenne
- Affiche une appréciation adaptée :
  - Moyenne > 16 → Excellent travail 🎉
  - Moyenne entre 11 et 16 → Bon travail 👍
  - Moyenne <= 10 → Courage, tu peux t’améliorer 💪
 
## Comment utiliser
1. Télécharger le fichier `moyenne_notes.py`
2. Lancer avec Python
3. Le programme affichera la somme, la moyenne et une appréciation
